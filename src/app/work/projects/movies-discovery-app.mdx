## Overview

The Movies Discovery App is a feature-rich web application that allows users to explore, search, and track movies. Built with the MERN stack and integrated with The Movie Database (TMDB) API, it provides an engaging platform for movie enthusiasts to discover new content, read reviews, and manage their watchlists.

## Key Features

### 1. Movie Discovery

- **Trending Movies**: Display currently trending movies
- **Popular Movies**: Most popular movies across platforms
- **Top Rated**: Highest-rated movies by users
- **Upcoming Releases**: Preview upcoming movie releases
- **Genre Filtering**: Browse movies by genre

### 2. Search & Filter

```javascript
// Advanced search with filters

  query: "inception",
  year: 2010,
  genre: "sci-fi",
  sort_by: "popularity.desc",

```

- **Intelligent Search**: Real-time search with autocomplete
- **Multi-criteria Filtering**: Genre, year, rating, popularity
- **Sort Options**: By popularity, rating, release date, title
- **Pagination**: Infinite scroll for seamless browsing

### 3. Movie Details

- **Comprehensive Information**: Plot, cast, crew, runtime, budget
- **Trailers & Videos**: Embedded YouTube trailers
- **User Ratings**: Display TMDB ratings and reviews
- **Similar Movies**: Recommendations based on current movie
- **Cast & Crew**: Detailed information about actors and directors

### 4. User Features

- **Personal Watchlist**: Save movies to watch later
- **Favorites**: Mark favorite movies
- **Viewing History**: Track watched movies
- **User Reviews**: Write and share movie reviews
- **Ratings**: Rate movies on a 10-point scale

### 5. Authentication System

- **User Registration**: Sign up with email and password
- **Login/Logout**: Secure JWT-based authentication
- **Password Reset**: Email-based password recovery
- **Profile Management**: Update user information and preferences
- **Protected Routes**: Authenticated-only features

### Backend (Node.js + Express)

```javascript
// TMDB API integration
router.get("/movies/trending", async (req, res) => {
  try {
    const response = await axios.get(`${TMDB_BASE_URL}/trending/movie/week`, {
      params: {
        api_key: process.env.TMDB_API_KEY,
        page: req.query.page || 1,
      },
    });
    res.json(response.data);
  } catch (error) {
    res.status(500).json({ error: "Failed to fetch movies" });
  }
});

// User watchlist endpoint
router.post("/watchlist", authenticateJWT, async (req, res) => {
  const { movieId } = req.body;
  const userId = req.user.id;

  const watchlistItem = new Watchlist({ userId, movieId });
  await watchlistItem.save();

  res.json({ message: "Added to watchlist", item: watchlistItem });
});
```

- **RESTful API**: Clean endpoint structure
- **TMDB API Integration**: Fetch movie data, trailers, cast
- **JWT Authentication**: Secure user sessions
- **MongoDB Integration**: Store user data, watchlists, reviews
- **Error Handling**: Comprehensive error middleware
- **Rate Limiting**: Prevent API abuse

## Challenges & Solutions

### Challenge 1: TMDB API Rate Limits

**Problem**: Hitting API rate limits with multiple users
**Solution**: Implemented Redis caching layer and request queuing

### Challenge 2: Image Loading Performance

**Problem**: Slow loading of high-res movie posters
**Solution**: Progressive image loading with blur placeholders

### Challenge 3: Real-time Search

**Problem**: Too many API calls during search
**Solution**: Debouncing with 300ms delay + cancel previous requests

### Challenge 4: Responsive Grid Layout

**Problem**: Inconsistent card sizes on different screens
**Solution**: CSS Grid with auto-fill and minmax for flexibility
